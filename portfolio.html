    3D Modeling

    Over the course of my time at UF, I had the opportunity to learn modeling 3D objects in Maya. We had several homework assignments which involves modeling a certain object or scene. We even had a homework that lets us practice applying texture onto the object correctly. For my final project, we were allowed to model anything as long as it combines what we learned in class and not simple such as a box. I decided to model a carousel because I love colorful things.

    3D Animation

    For my 3D animation class, we were given a choice of using Maya or Blender as our program for animation and I decided to use Blender since I learned modeling in Maya. I learned how to animate objects frame by frame. In the image shown, our goal was to create different expressions on a given head model. For our final project, I made a small animation of a brief encounter between a lost cat and a boy at the park.

    Coding

    I was able to learn about the Java Swing and its features. We learned the GUI components and use what we learned in class to make our final project. For my final project, I decided to make a Firework Simulator. I wanted different features for the user to manipulate, such as the color of the firework and the pattern it takes on. The user would choose a background picture from the "File" tab and can begin clicking anywhere on the picture with the cursor. A stream will appear from the bottom and explode into a firework depending on where you clicked. Overall, I feel pretty accomplished about this project.

    Games

    I was in a group of 4, including myself. We had to create a game from scratch and the only requirement for the game is that it must be a narrative. We were inspired by Bioshock Infinite and decided to set the story in the sky. Basically in the game the user plays as a recruit for a company and explores around the sky of a moon called Caelum VII to collect resources that will allow the recolonization of the human race. This game was created on the game engine Unity and everything in it is made by the team.
